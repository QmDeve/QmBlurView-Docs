# 概览
`QmBlurView Components` 是一个功能强大且灵活的 Android 库，提供多种模糊效果和组件，以增强您的应用程序的视觉吸引力。无论您是想添加细微的背景模糊还是更明显的效果，该库都能满足您的需求。

该库以性能为核心，利用本地 C++ 代码确保高效处理模糊效果，同时不影响速度或响应能力。通过一系列可定制的选项，您可以轻松调整模糊效果以适应您的应用程序的设计和用户体验需求。
