QmBlurViewБиблиотека компонентов Gaussian Blur для Android

Начинай быстро
View on GitHub
Скачать Demo

Высокая производительность

Использование нативного алгоритма размытия, реализованного на C/C++, обеспечивает максимальную скорость и плавность.

Разнообразные компоненты

Включает в себя различные компоненты с эффектом размытия.

Поддержка загрузки изображений

Встроенная поддержка преобразований для Glide и Picasso позволяет легко добавлять эффекты размытия к изображениям.

Простота интеграции

Простые XML-атрибуты и Java/Kotlin API. Легко интегрируется в существующие проекты.
